Will Level 3 radar data be available?

No. For more information regarding climate data availability, please refer to the MET Malaysia FAQ.

Why are some mappings missing?

Government data is text-only. We manually map regions based on descriptions, which can cause slight mismatches with official boundaries.

Why use RainViewer instead of Windy?

RainViewer provides a free API for radar data, whereas Windy does not support the specific map overlays we need. More MET Malaysia information on data in a future update.
